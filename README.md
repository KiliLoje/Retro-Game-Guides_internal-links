# BDR Game Guides


### !!! THE PROJECT GOT ENTIERLY REMADE. I'M KEEPING THIS FOR ARCHIVING PURPOSES !!!

A collection of comprehensive game guides for retro achievement hunting.

## Features

- Interactive checklists for tracking progress
- Multiple themes and color palettes
- Tabbed guide organization
- Local storage for saving progress
- Guide builder for creating new guides

## Getting Started

1. Visit the [Retro Game Guides](https://bigdonrob.github.io/Retro-Game-Guides/)
OR clone this repository and open `index.html` in your web browser
2. Browse available guides to use or build your own
3. Submit your own guide to be added to the collection

## Building Guides

Use the `builder.html` tool to create new game guides:
- Set game metadata (ID, title, theme, palette)
- Add tabs with different panel types
- Export as ZIP for submission
